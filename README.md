# Agent-DDL

AI Agent competition deadline tracker for hackathons, benchmark challenges, and academic evaluation tracks.

- Live site: https://just-agent.github.io/agent-ddl/
- Repository: https://github.com/Just-Agent/agent-ddl
- Stack: React 19, TypeScript, Vite, Tailwind CSS

## Scripts

```bash
npm ci
npm run build
npm run preview
```

## Deployment

GitHub Pages is deployed from `.github/workflows/deploy-pages.yml`.
